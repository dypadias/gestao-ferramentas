# Gestão de Ferramentas - Colafix

Sistema desktop para controle de inventário e rastreabilidade de ferramentas industriais.

## 🚀 Status do Projeto
Versão MVP funcional com arquitetura baseada em Python e Flet.

## ✨ Funcionalidades Atuais
- **Dashboard de Controle:** Monitoramento de ferramentas disponíveis, emprestadas e listagem de atrasos.
- **Registro de Empréstimos:** Interface com validação de campos e seleção de datas via calendário nativo.
- **Gestão de Devoluções:** Baixa de itens pendentes com um clique e atualização automática de status.
- **Painel de Cadastros:** Registro e visualização de ferramentas (com patrimônio), funcionários e setores.

## 🛠️ Tecnologias
- **Linguagem:** Python 3.14
- **Interface:** Flet (Material Design)
- **Dados:** SQLite

## ⚙️ Como Rodar
1. Instale o Flet: `pip install flet`
2. Execute: `python app.py`
